Set up

It is easy to install Fotorama:

<!-- 1. Link to the newest jQuery, -->
<script src="http://code.jquery.com/jquery-1.9.1.min.js"></script>

<!-- fotorama.css & fotorama.js. -->
<link  href="http://fotorama.s3.amazonaws.com/4.0.7/fotorama.css" rel="stylesheet">
<script src="http://fotorama.s3.amazonaws.com/4.0.7/fotorama.js"></script>

<!-- 2. Add images to <div class="fotorama"></div>. -->
<div class="fotorama">
    <img src="1.jpg">
    <img src="2.jpg">
    <img src="3.jpg">
</div>

<!-- 3. Enjoy! -->

Fotorama’s CDN is provided by Amazon S3, feel free to use it:

<link  href="http://fotorama.s3.amazonaws.com/4.0.7/fotorama.css" rel="stylesheet">
<script src="http://fotorama.s3.amazonaws.com/4.0.7/fotorama.js"></script>

Or download zipped Fotorama, unpack and host it on your server if you want:

fotorama-4.0.7.zip
